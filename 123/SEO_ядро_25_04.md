# SEO-ядро и стратегия: темлячные бусины, брелоки и подвески 

Рабочий документ для сайта 102procenta.ru. Версия: 25 апреля 2026. Подготовлено по результатам обсуждения, Wordstat-выгрузок и текущей SEO-логики блога. 

# 1. Главный вывод 

В текущей структуре нельзя смешивать все назначения бусин в один одинаковый блок. Нужно развести не по почти одинаковым названиям, а по сценариям использования: темляки, брелоки/ключи, подвески на сумку/телефон/молнию. 

• Категория «Темлячные бусины» остается основной товарной категорией: у проекта много товаров, более 20 уникальных моделей. 

• «Бусины для темляка» не удаляется, но используется как вспомогательная SEO-посадочная и анкор в текстах. 

• «Бусины для ножа» не нужно выводить отдельной карточкой рядом с «Темлячными бусинами» в коммерческом блоке, потому что для пользователя это близкие сценарии. 

• В коммерческих блоках лучше использовать три сценарных карточки: темлячные бусины, брелоки/ключи, подвески на сумку/телефон/молнию.

# 7. Что дать OpenCode 

Команда для внедрения в сайт: 

1. Во всех article-commercial-block заменить карточки так, чтобы не было смысловых дублей: «Темлячные бусины», «Бусины для брелоков и ключей», «Бусины для подвесок на сумку и телефон». 2. Не удалять страницы /businka/categories/temlyachnye-businy/, /businka/categories/businy-dlya-temlyaka/ и /businka/categories/businy-dlya-nozha/. 3. «Темлячные бусины» оставить основной товарной категорией. 4. «Бусины для темляка» использовать в текстах и анкорах как вспомогательный SEO-запрос. 5. «Бусины для ножа» использовать там, где есть ножевой контекст, но не выводить отдельной карточкой рядом с «Темлячными бусинами» в каждом блоке. 6. Не делать коммит без подтверждения. После изменений показать git diff --stat и git diff --check. 

# 8. План дальнейших страниц 

• Создать или усилить посадочную «Темлячные бусины» как главную товарную категорию. 

• Оставить «Бусины для темляка» как объясняющую страницу: как выбрать отверстие, вес, материал и сценарий использования. 

• Позже создать посадочную «Бусины для брелоков и подвесок», если будет достаточно товаров и контента. 

• В статьях продолжать воронку: паракорд - техника - темляк - темляк с бусиной - темлячные бусины. 

• Для Wordstat собирать частотность отдельно по кластерам: ключи, сумка, телефон, молния, темляки.
